  • 라우팅 설정

    라우팅 설정 관련해서는 https://reactrouter.com/web/guides/quick-start 사이트를 참조하세요.

    1. Home, Memo, Trash 컴포넌트를 만든다
    2. App 컴포넌트를 만들다
      1. 라우팅 기능을 추가 한다. BrowserRouter, Switch, Route, Redirect 컴포넌트
      2. 메뉴 기능을 추가한다. Link, NavLink 컴포넌트
    3. / 요청으로 들어오면: Home 컴포넌트를 렌더링 한다
    4. /memo 요청으로 들어오면: Memo 컴포넌트를 렌더링 한다

      메모장 화면에서는 보유한 메모 목록을 보여준다

    5. /trash 요청으로 들어오면: Trash 컴포넌트를 렌더링 한다

      삭제한 메모는 휴지통 화면에서 확인할 수 있다

    6. 위 세 개 규칙을 벗어나면: 루트(/) 요청으로 리다이렉션 한다
    7. /memo 요청이 들어오면: 좌측에 메모 목록을 보여준다
    8. /memo/* 요청이 들어오면: 우측에 뭔가를 표시한다
    9. /memo/add 요청시: AddMemo 컴포넌트를 렌더링 한다
    10. /memo/:id 요청시: Memo 컴포넌트를 렌더링 한다
    11. / 요청시: '메모가 없습니다'를 렌더링 한다
                    
# src/routes/RouterHome.js
<BrowserRouter>
    <Switch>
        <Route path="/" exact component={Home} />
        <Route path="/memo" component={Memo} />
        <Route path="/trash" component={Trash} />
        <Redirect path="*" to="/" />
    </Switch>
</BrowserRouter>


# src/routes/RouterMemo.js
<Switch>
    <Route path={`${match.url}/add`} exact component={AddMemo} />
    <Route path={`${match.url}/:id`} exact component={Memo} />
    <Route path={`${match.url}/`} exact component={() => <div>메모가 없습니다. 새로운 메모를 만들어 보세요.</div>} />
</Switch>


# src/routes/RouterTrash.js
<Switch>
    <Route path={`${match.url}/:id`} exact component={RemovedMemo} />
    <Route path={`${match.url}/`} exact component={() => <div>휴지통이 비었습니다.</div>} />
</Switch>

Prettier 설정 Prettier는 코드 포맷터(Code Formatter)다 Prettier는 소스 코드를 깃에 커밋할 때 정의한 규칙에 따라 코드의 형식을 자동으로 수정한다. Prettier 모듈을 설치할 때는 --save-exact 옵션을 사용하세요. --save-exact 은 Prettier 버전이 달라져도 코드 스타일 변화가 생기지 않도록 하는 옵션입니다. $ npm install --save-dev --save-exact prettier eslint-plugin-prettier: 코드 코맷할 때 Prettier를 사용하게 만드는 규칙을 추가합니다. $ npm install --save-dev eslint-plugin-prettier eslint-config-react: react 프로젝트에 맞춰진 ESLint 규칙을 추가합니다 $ npm install --save-dev eslint-config-react eslint-config-prettier: Prettier와 충돌할 설정들을 비활성화합니다. $ npm install --save-dev eslint-config-prettier .prettierrc 파일에 아래 내용을 추가 $ .prettierrc { "printWidth": 130, "tabWidth": 2, "singleQuote": true, "semi": true, "useTabs": false, "trailingComma": "all", "proseWrap": "always", "requireConfig": false, "bracketSpacing": true, "jsxBracketSameLine": false, "jsxSingleQuote": false, "quoteProps": "as-needed", "endOfLine": "crlf" } 4-3. eslint 설정 eslintrc.json 파일에 아래 내용을 추가 https://poiemaweb.com/eslint https://eslint.org/docs/user-guide/configuring/rules $ eslintrc.json { "env": { "browser": true, "commonjs": true, "es2021": true, "node": true, "jest": true, "es6": true }, "extends": [ "react-app", "prettier", "airbnb" ], "parser": "babel-eslint", "parserOptions": { "ecmaVersion": 12 }, "globals": { "_": false }, "plugins": ["import", "html"], "rules": { // windows linebreaks when not in production environment "linebreak-style": [ "error", "unix" ], "react/jsx-filename-extension": [ 1, { "extensions": [ ".js", ".jsx" ] } ], // "off" or 0 -
